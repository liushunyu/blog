python 文件读写模式 r, r+, w, w+, a, a+ 的区别。



## 区别

| 模式 | 可做操作 | 若文件不存在 | 是否覆盖   |
| ---- | -------- | ------------ | ---------- |
| r    | 只能读   | 报错         | -          |
| r+   | 可读可写 | 报错         | 是         |
| w    | 只能写   | 创建         | 是         |
| w+   | 可读可写 | 创建         | 是         |
| a    | 只能写   | 创建         | 否，追加写 |
| a+   | 可读可写 | 创建         | 否，追加写 |

- 若目录不存在则不会自动创建文件，并报错



## 操作

```python
import os

# os.linesep 代表当前操作系统上的换行符
with open('test.txt', mode='w') as f:
    f.write('abc' + os.linesep)
    f.writelines(['1\n', '2\n', '3\n'])

with open('test.txt', mode='r') as f:
    r = f.readlines()
```
