python 文件读写模式 r, r+, w, w+, a, a+ 的区别。



## 区别

| 模式 | 可做操作 | 若文件不存在 | 是否覆盖   |
| ---- | -------- | ------------ | ---------- |
| r    | 只能读   | 报错         | -          |
| r+   | 可读可写 | 报错         | 是         |
| w    | 只能写   | 创建         | 是         |
| w+   | 可读可写 | 创建         | 是         |
| a    | 只能写   | 创建         | 否，追加写 |
| a+   | 可读可写 | 创建         | 否，追加写 |



## 操作

```python
def file_operation():
    with open('test.txt', mode='r') as f:
        f.write('abc')
        r = f.readlines()
        print r
        print '---done---'

file_operation()
```
