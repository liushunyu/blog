PyCharm 新建文件时自动添加基础信息。



## 自动添加基础信息步骤

> PyCharm -> Preferences -> Editor -> File and Code Templates -> Python Script -> 添加以下代码



```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
# @Time    : ${DATE} ${TIME}
# @Author  : Shunyu Liu
# @Project : ${PROJECT_NAME}
# @File    : ${NAME}
# @Software: ${PRODUCT_NAME}
"""

"""
```
