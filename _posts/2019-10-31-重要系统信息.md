查看重要系统信息。



## 跑代码必看

gpu 信息

```bash
watch -n 1 nvidia-smi
# watch -n 1 是每一秒刷新一次
```



cpu、内存、进程信息

```
htop
```



## 配置必看

conda 配置信息

```bash
conda config --show
```
